# Markdown Editor (Python + Tkinter + ttkbootstrap)

A modern Markdown Editor with:

- Live HTML preview
- Dark/Light theme toggle
- Toolbar with formatting buttons
- Sidebar file browser
- Export to HTML and PDF
- Packaged as standalone `.exe`
